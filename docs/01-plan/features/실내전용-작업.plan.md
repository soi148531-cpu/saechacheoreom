# 실내 전용 작업 기록 및 협상 가격 입력 기능

- Feature: 실내전용-작업
- Created: 2026-04-12
- Phase: Plan

---

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 실외 세차 없이 실내 청소만 진행할 때 앱에 기록할 방법이 없고, 협상 가격(예: 20,000원)을 입력할 수 없어 매출 누락 가능성이 있다 |
| **Solution** | 세차 완료 처리 시 "작업 유형" 선택 추가 (외부 세차 / 실내 전용 / 둘 다) — 실내 전용 선택 시 가격 직접 입력 가능 |
| **Function UX Effect** | 직원 앱에서 실내 전용 체크 → 가격 입력 one-step 처리, 관리자 화면에서 배지로 구분 |
| **Core Value** | 모든 작업 유형이 앱에 기록되어 청구 누락 방지, 협상 내역 이력 관리 |

## 1. 현황 분석

### 현재 작업 유형 체계

```
wash_records.service_type
  'regular'   → 외부 세차만
  'interior'  → 외부 + 실내 추가 (+10,000원)
  (신규 필요)  → 실내 전용 (협상 가격)
```

### 현재 CompletionModal 동작

1. 직원이 세차 완료 버튼 클릭
2. hasInterior 체크박스 → 실내청소 체크 시 +10,000원 자동 추가
3. service_type = 'interior' 또는 'regular' 로 저장

### 문제점

- 실외 없이 실내만 진행하는 케이스 처리 불가
- 가격이 기본 단가 + 고정 10,000원으로만 설정됨
- 고객과 협상한 임의 금액(예: 20,000원 단독) 입력 방법 없음

## 2. 요구사항

### 필수 요구사항

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| R1 | 세차 완료 처리 시 작업 유형 선택 (외부 세차 / 실내 전용 / 외부+실내) | HIGH |
| R2 | "실내 전용" 선택 시 가격 직접 입력 가능 (기본값: app_settings의 interior 가격, 현재 20,000) | HIGH |
| R3 | 직원 앱 — 실내 전용 완료 처리 후 wash_records에 service_type='interior_only' 로 저장 | HIGH |
| R4 | 관리자 고객 상세 페이지 — 실내 전용 세차 기록에 "실내" 배지 표시 | MEDIUM |
| R5 | 청구 페이지 — 실내 전용 항목도 청구 금액에 포함 (이미 price 기준으로 집계됨) | MEDIUM |

### 선택 요구사항

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| R6 | 관리자 직접 실내 전용 기록 추가 (고객 상세 페이지) | LOW |
| R7 | 실내 전용 기본 가격을 settings 페이지에서 변경 (이미 가격관리 시스템으로 지원됨) | LOW |

## 3. 데이터 모델

### DB 변경사항 — 없음

기존 `wash_records.service_type` 컬럼의 허용 값만 확장 (문자열 필드, ENUM 아님):

```
기존: 'regular' | 'interior'
신규: 'regular' | 'interior' | 'interior_only'  ← 신규 추가
```

`price` 컬럼이 이미 있으므로 협상 금액 그대로 저장 가능. **마이그레이션 불필요**.

---

## 4. 서비스 유형 정의 (확정)

| service_type | 의미 | 가격 결정 방식 |
|--------------|------|----------------|
| `regular` | 외부 세차만 | 차량 단가 (unit_price) |
| `interior` | 외부 + 실내 | 단가 + app_settings.interior (기본 10,000) |
| `interior_only` | 실내 전용 | 직원/사장 입력 (기본값: app_settings.interior = 20,000) |

## 6. UI 설계

### CompletionModal — 작업 유형 선택 영역

```
[작업 유형]
● 외부 세차        → 현재와 동일
○ 실내 전용        → 가격 입력창 노출 (기본: 20,000원)
○ 외부 + 실내 세차  → 단가 + 실내가격 자동 계산

// 실내 전용 선택 시
실내 작업 금액: [ 20,000 ]원  ← 편집 가능
(고객 협상 시 금액을 변경하세요)
```

### 고객 상세 페이지 — 세차 기록 배지

```
2026-04-10  [실내전용]  20,000원  메모: 고객 요청으로 내부만
2026-04-03  [외부+실내] 45,000원
2026-03-27  [외부]      35,000원
```
